# Архив документации (OldDocs)

**Не использовать** этот каталог как источник истины для разработки и handoff без явного запроса.

Здесь лежат **прежние версии** документов и технические задания. Актуальный канон — **модульная документация** в `docs/00_core/`, `docs/10_operations/`, `docs/20_api/`, `docs/30_flutter/`, `docs/90_current/` и указатель **`docs/README_MODULAR.md`**.

Прежние четыре монолитных файла из корня `docs/` перенесены в **`legacy_monolith/`** для сверки или восстановления контекста.

Если понадобится что-то из этого архива — откройте нужный файл здесь или перенесите актуальные правки в соответствующий модульный файл в `docs/`.
